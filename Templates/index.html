Pets DB


#Get all the pets
See All the pets

  • It uses the GET method to get all the pets
  • It returns a JSON object with all the pets
  • It has route: /api/pet

#Add pets do database

  • It uses the POST method to add pets to database
  • It returns a string with the message "Pets added to database"
  • It has route: /api/pet

#Get a pet by id



  • It uses the GET method to get a pet by id.
  • It returns a JSON object with the pet
  • It has route: /api/pet/{id}

#Update a pet by id

*The name of the pet with respective ID changes to Tomy on clicking it.




  • It uses the PATCH method to update a pet by id.
  • It returns a string with the message "Pet updated"
  • It has route: /api/pet/{id}

#Delete a pet by id



  • It uses the DELETE method to delete a pet by id.
  • It returns a string with the message "Pet deleted"
  • It has route: /api/pet/{id}
